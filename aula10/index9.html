FAQ

O que é HTML?

HTML é a linguagem de marcação usada para estruturar páginas na web.

Como usar tags semânticas?

Use tags como <header>, <main> e <footer> para indicar a função do conteúdo.

O que é CSS?

CSS é a linguagem usada para estilizar páginas HTML, controlando cores, layouts e fontes.

Como tornar um site acessível?

Use markup semântico, alt em imagens, e teste com leitores de tela.

O que é SEO?

SEO (Search Engine Optimization) é o conjunto de práticas para melhorar o posicionamento do site em buscadores.
